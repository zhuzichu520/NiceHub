### 介绍:
一个开源的Github Android 客户端项目，开发目的主要是为了学习Kotlin语法以及MVVM架构，项目中所用的技术有Kotlin+buildSrc+MVVM+DataBinding+Navigation+ViewModel+LiveData+Lifecycle+Material等等,项目现在处于每天更新之中，会越来越丰富哦。。。
