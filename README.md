### 介绍:
一个开源的Github Android 客户端项目，开发目的主要是为了学习Kotlin语法以及MVVM架构，项目中所用的技术有Kotlin+buildSrc+MVVM+DataBinding+Navigation+ViewModel+LiveData+Lifecycle+Material+RxJava+Dagger2等等。
